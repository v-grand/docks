# Обзор платформы

Добро пожаловать в документацию нашего проекта! Эта платформа представляет собой комплексное решение для разработки, развертывания и управления микросервисами.

## Ключевые компоненты

Платформа состоит из нескольких ключевых компонентов, каждый из которых находится в своем репозитории и управляется как субмодуль:

-   **Сервисы**: Основные бизнес-приложения (`auth`, `api`, `frontend`).
-   **Инфраструктура**: Terraform-модули и конфигурации для облачных провайдеров (`infra-aws`, `infra-gcp`).
-   **Шаблоны**: Стандартизированные шаблоны для быстрого создания новых компонентов.

Более подробную информацию о шаблонах можно найти в разделе [Шаблоны](templates.md).
