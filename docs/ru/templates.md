# Шаблоны проектов

Этот раздел содержит описание стандартизированных шаблонов, используемых для создания новых компонентов платформы.

---

### `template-service`

**Репозиторий:** `https://github.com/v-grand/template-service.git`

Этот шаблон предназначен для быстрого создания новых микросервисов. Он включает в себя:

-   Базовую структуру приложения на Python.
-   Готовый `Dockerfile` для сборки сервиса.
-   Примеры конфигурации для локального запуска и продакшена.
-   Настроенный CI/CD пайплайн.

---

### `template-platform`

**Репозиторий:** `https://github.com/v-grand/template-platform.git`

Шаблон для создания полноценной платформы "с нуля". Он объединяет в себе все остальные компоненты и служит отправной точкой для развертывания всего стека. Включает:

-   Структуру субмодулей для всех сервисов и инфраструктурных репозиториев.
-   `docker-compose.yml` для локальной разработки.
-   `Makefile` с командами для управления проектом.

---

### `template-docks`

**Репозиторий:** `https://github.com/v-grand/template-docks.git`

Шаблон для создания сайта с документацией, подобного этому. Он включает:

-   Настроенный `MkDocs` с темой `material`.
-   Примеры структуры документации.
-   Готовый воркфлоу для публикации сайта на GitHub Pages.
